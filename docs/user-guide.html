Working with Notes

  1. Go to customer's Notes tab
  2. Type your note in the text area
  3. Select note type from dropdown
  4. Click Add Note
üí° Pin Important Notes
Click the pin icon on any note to highlight it. Pinned notes appear in the Overview tab for quick reference.

Managing Credits

Credits allow customers to pre-pay for sessions (package deals).

  1. Click Adjust Credits in the sidebar
  2. Select Add or Subtract credits
  3. Enter amount and reason
  4. Click Update Credits

Managing Flags

Flags help you track important follow-ups and never miss a customer touchpoint.

  1. Click the flag button in the sidebar
  2. Select flag type (Follow Up, Payment Due, etc.)
  3. Enter your message
  4. Set trigger date (optional)
  5. Click Add Flag

7. Providers

Access via GuidePost ‚Üí Providers

Setting Working Hours

  1. Click Edit on the provider
  2. Go to Working Hours section
  3. For each day, toggle on/off and set start/end times
  4. Click Save Changes

Managing Days Off

For vacations and holidays:

  1. Go to provider's Days Off tab
  2. Click Add Days Off
  3. Select start and end dates
  4. Enter reason (optional)
  5. Click Save
üìù Note
Days off automatically block all bookings for that provider during the specified period.

10. Troubleshooting

Common Issues

Buttons Not Working

  1. Clear your browser cache
  2. Refresh the page
  3. Check browser console for JavaScript errors
  4. Ensure no plugin conflicts exist

Emails Not Sending

  1. Check SMTP settings are correct
  2. Use the "Send Test Email" feature
  3. Check spam/junk folders
  4. Review email logs for error messages

Calendar Not Loading

  1. Ensure JavaScript is enabled
  2. Check for browser extensions blocking scripts
  3. Try a different browser
  4. Clear browser cache
